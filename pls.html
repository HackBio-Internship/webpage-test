Python and R for Life Sciences

Python and R are the most demanded programming languages in bioinformatics. This course is a deep dive into coding for life sciences. We end with a quick project that brings your biopython and bioconductor (R) skills to life.

Level: Novice, Enthusiasts, no degree needed.

Duration: 8-10 weeks (5hrs/week)

Enroll Now Download Syllabus

Python and Biopython

Conceptual and practice based introduction to python from a biologist point of view. Libraries include biopython, pandas & seaborn.

R, R Studio and Bioconductor

Conceptual and practice based introduction to R and R Studio from a biologist point of view. The main package of focus is bioconductor.

Statistical Programming and Visualization

Learn statistical validation and data visualization with R (bioconductor, ggplot2, rbase) and Python (pandas & seaborn).
